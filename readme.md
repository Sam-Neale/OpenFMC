# OpenFMC

OpenFMC is an open, desktop-based Flight Management Computer (FMC) and flight operations platform for Infinite Flight. Built with Electron, OpenFMC connects directly to a running Infinite Flight session through the ConnectV2 API, providing advanced flight planning, navigation management, and aircraft guidance capabilities beyond those available natively.

The project aims to deliver a modern FMC experience inspired by real-world airline avionics while remaining simulator-agnostic in its internal architecture and extensible for future integrations.
